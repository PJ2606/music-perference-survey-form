# music-perference-survey-form
# Survey Website Project

A simple **survey website** built using **HTML**.  
This project is a beginner-friendly practice project to understand **HTML forms**, user input, and basic web development.

## Features
- User-friendly survey form
- Multiple input types: text, radio buttons
- Clean and simple layout

## Purpose
Created as a **practice project to learn:
- HTML form elements
- Basic webpage structure
- Collecting user responses

## How to View
The project is hosted live on GitHub Pages:

[View Live Survey](https://pj2606.github.io/music-perference-survey-form/)

## Usage
1. Open `index.html` in a web browser to view the survey locally.  
2. Fill out the form to see how user input works.
